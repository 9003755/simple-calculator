```typescript
// 正弦定理：a/sinA = b/sinB = c/sinC
const sideB = (sideA * Math.sin(bRad)) / Math.sin(aRad);
const sideC = (sideA * Math.sin(cRad)) / Math.sin(aRad);
```

#### GPS距离计算
```typescript
// Haversine公式计算大圆距离
const a = Math.sin(dLat / 2) * Math.sin(dLat / 2) +
          Math.cos(lat1Rad) * Math.cos(lat2Rad) *
          Math.sin(dLon / 2) * Math.sin(dLon / 2);
const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1 - a));
const distance = R * c; // R = 6371km (地球半径)
```

## 🎯 使用说明

### 基础计算
1. 选择"基础计算"模式
2. 使用数字按钮输入数值
3. 选择运算符进行计算
4. 点击"="查看结果
5. 使用"x²"和"√x"进行高级运算

### 三角形计算
1. 选择"三角形计算"模式
2. 输入角A和角B的度数
3. 输入已知边长
4. 选择已知边对应的角(A/B/C)
5. 点击"计算三角形"获取结果

### GPS距离计算
1. 选择"GPS距离"模式
2. 选择坐标格式(度/度分秒)
3. 输入两个点的经纬度
4. 点击"计算距离和航向"
5. 查看距离和双向航向角

### 速度单位换算
1. 选择"速度换算"模式
2. 输入速度值
3. 选择输入单位
4. 选择输出单位
5. 点击"转换速度"获取结果
