# 多功能计算器 - 项目文档

## 📋 项目概览

本项目是基于《多功能计算器软件需求文档（SRD）》开发的一款现代化Web计算器应用，专注于三角函数计算和科学计算功能。

## 🎯 核心功能

### 计算功能
- ✅ **基本运算**：加、减、乘、除、百分比
- ✅ **三角函数**：sin、cos、tan、cot、sec、csc
- ✅ **反三角函数**：arcsin、arccos、arctan、arccot、arcsec、arccsc
- ✅ **双曲函数**：sinh、cosh、tanh
- ✅ **科学函数**：ln、log、sqrt、exp、阶乘
- ✅ **角度模式**：度(°)、弧度(rad)、梯度(grad)

### 界面功能
- ✅ **函数图像绘制**：可视化数学函数
- ✅ **计算历史**：记录和查看历史计算
- ✅ **主题切换**：深色/浅色/系统主题
- ✅ **键盘快捷键**：完整的键盘操作支持
- ✅ **响应式设计**：适配各种屏幕尺寸

## 🏗️ 技术架构

### 前端技术
- **React 18** - 现代化UI框架
- **TypeScript** - 类型安全的JavaScript
- **Tailwind CSS** - 实用优先的CSS框架
- **Zustand** - 轻量级状态管理
- **Vite** - 快速构建工具

### 项目结构
```
src/
├── components/          # React组件
├── lib/                # 核心计算库
├── stores/             # 状态管理
├── hooks/              # 自定义Hook
└── __tests__/          # 测试文件
```

## 📖 使用指南

### 基本操作
1. **数字输入**：点击数字按钮或使用键盘
2. **基本运算**：使用运算符按钮
3. **三角函数**：输入角度后点击函数按钮
4. **角度切换**：使用角度模式选择器

### 高级功能
1. **函数图像**：点击"显示函数图像"按钮
2. **主题切换**：右上角主题选择器
3. **键盘快捷键**：查看设置面板了解快捷键
4. **计算历史**：查看历史记录面板

### 快捷键列表
- `0-9`：数字输入
- `+ - * /`：基本运算
- `Enter/=`：计算结果
- `Shift+S/C/T`：sin/cos/tan
- `Esc/C`：清除
- 更多快捷键详见应用内帮助
